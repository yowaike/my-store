# Интернет-магазин (Контрольная работа №2)

## Описание
Веб-приложение для управления интернет-магазином с системой аутентификации, JWT-токенами и разграничением прав доступа (RBAC).

## Функциональность
- Регистрация и аутентификация пользователей
- JWT access + refresh токены
- Управление товарами (CRUD)
- Управление пользователями (для администратора)
- Ролевая модель доступа

## Технологии

### Backend:
- Node.js + Express
- bcrypt, jsonwebtoken
- Swagger (документация API)

### Frontend:
- React.js
- Axios, React Router DOM
- SCSS
