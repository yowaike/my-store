# Интернет-магазин (Контрольная работа №1)

## Описание
Веб-приложение интернет-магазина с CRUD операциями для товаров.

## Стек технологий
- **Frontend:** React, SASS, Axios
- **Backend:** Node.js, Express, Swagger
- **База данных:** In-memory массив

## Запуск проекта

### Бэкенд:
```bash
cd server
npm install
node index.js
